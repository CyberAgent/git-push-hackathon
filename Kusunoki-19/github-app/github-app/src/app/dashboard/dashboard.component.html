MyWebHub の使用方法

MyWebHubへようこそ!

MyWebHubは、GithubのAOuth認証機能をつかって、Gistの一覧表示や、投稿が行えるアプリケーションです。

MyWebHubを使用するには、OAuth認証を行いアクセストークンを取得する必要があります。
OAuth認証を行うにはOAuth認証のページに行き、「OAuth認証を行う」を押してください。
そうすると、Githubへ移動し、ログイン画面が現れます。このとき、このアプリケーションが使用する権限が表示されます。

また、OAuth認証で得たアクセストークンは、cookieに保存されています。 cookieからアクセストークンを削除したい場合は、OAuth認証のページ に行き、「アクセストークンの削除」を押してください。
アクセストークンが有効であるかは、 右上のマーク/で確認できます。
